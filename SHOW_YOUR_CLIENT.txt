Q: How often does it sync?
A: Once per day automatically at midnight UTC, plus manual on-demand.

Q: What if an API fails?
A: 3 automatic retries with exponential backoff. Syncs continue with 
   other sources. Error logged.

Q: How many jobs?
A: ~700 per run (150 Adzuna, 500 USAJobs, 45 Greenhouse).

Q: Duplicates?
A: Zero. Composite key (source + externalJobId) prevents duplicates.
   We proved it by syncing twice - second run had 0 new inserts.

Q: What's the API?
A: POST /sync/all (or ?source=X for single). GET /status for metrics.
   Full docs in the testing guide.

A: Directly via MongoDB (we showed the collection structure).
   Search API can be added in Phase 2.

   We'll monitor and alert.

   Now: Automatic, 10 seconds, zero duplicates, 700 jobs daily
